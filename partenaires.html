CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. It is used to define the layout, colors, fonts, and other visual aspects of a web page.

CSS Syntax

CSS syntax consists of a selector and a set of declarations, which are enclosed in curly braces ({ }). The selector specifies which HTML elements the declarations should apply to. Here's an example:

selector {
  property1: value1;
  property2: value2;
  property3: value3;
}

In this example, "selector" is the HTML element or elements that the declarations should apply to, "property1", "property2", and "property3" are the properties being defined, and "value1", "value2", and "value3" are the values of those properties.

CSS Selectors

There are several types of selectors in CSS, including:

  • Element selectors - selects HTML elements by their name (e.g. p { })
  • Class selectors - selects HTML elements by their class attribute (e.g. .my-class { })
  • ID selectors - selects HTML elements by their ID attribute (e.g. #my-id { })
  • Attribute selectors - selects HTML elements by their attribute values (e.g. [type="text"] { })
