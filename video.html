What is HTML?

HTML (Hypertext Markup Language) is a markup language used to create and structure content on the web. It consists of a series of tags that define the structure and content of a web page.

HTML Tags

Some of the most commonly used HTML tags include:

  • <html> - Defines the root of an HTML document.
  • <head> - Contains metadata about the document, such as the title and links to stylesheets.
  • <body> - Contains the content of the document, such as text and images.
  • <div> - Defines a division or section of the document.
  • <p> - Defines a paragraph of text.
  • <a> - Defines a hyperlink to another web page or resource.
  • <img> - Defines an image to be displayed on the page.
